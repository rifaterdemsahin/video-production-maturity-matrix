## Docker Compose

```yaml
version: '3.8'
services:
  qdrant:
    image: qdrant/qdrant:latest
    ports:
      - "6333:6333"
      - "6334:6334"
    volumes:
      - ./qdrant_storage:/qdrant/storage
    restart: unless-stopped

  ollama:
    image: ollama/ollama:latest
    ports:
      - "11434:11434"
    volumes:
      - ollama_data:/root/.ollama
    restart: unless-stopped
    deploy:
      resources:
        reservations:
          devices:
            - capabilities: [gpu]  # Remove if no GPU

volumes:
  ollama_data:
```

## Usage

```bash
# Start the stack
docker compose up -d

# Pull the model (first time)
docker exec -it <ollama-container> ollama pull nomic-embed-text

# Index all docs
python 5_Symbols/indexer.py

# Search
python 5_Symbols/search.py "how to improve editing workflow"
```
