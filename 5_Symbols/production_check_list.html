🎬 Video Production Checklist

Step-by-step guide for a flawless recording session

0 / 0 tasks completed

🖥️ Environment Setup

Single Monitor Mode Use multi-monitor tool to set single-monitor layout before starting.
Configure Stream Deck for Video Production Open Stream Deck and load the video production profile.
Load Script to Prompter Copy the latest script into the teleprompter software.
Set Claude as AI Assistant & Update Script Open Claude, set production context, refine and update the script.
Add Claude to Stream Deck Context Assign Claude shortcut in Stream Deck for quick video production queries.
Confirm Updated Script is Loaded Verify the prompter shows the final revised script before recording.

🎙️ OBS & Recording Configuration

Add OBS Shortcuts with Replay Buffer Configure hotkeys for record start/stop and replay buffer in OBS.
Turn Off Voice Assistant Disable Siri / Cortana / Google Assistant to avoid interruptions.
Set Video Output Folder in Stream Deck Confirm the destination folder shortcut is correct on Stream Deck.
Mitigate Production Errors Review known error list; resolve any OBS/scene/source issues before recording.
Verify OBS Record Button & Replay Buffer Start a test record to confirm both record and replay buffer are active.
Use OBS Projector for Confidence Monitor Enable fullscreen projector on the secondary display for live preview.
Side-by-Side Confidence Monitor Layout Arrange prompter and OBS preview in side-by-side mode.
Practical Record Shortcuts on Stream Deck Test all record shortcuts are mapped and responsive on Stream Deck.
Multi-Version Action Setup Prepare multi-take / multi-version recording strategy in OBS.
Check Recorded Videos with Timecodes Review recorded clip timestamps to confirm successful recordings.

📷 Camera Setup

Power On Sony ZV1 Camera Turn on the Sony ZV1 and confirm it connects to OBS / capture card.
Check Autofocus Confirm autofocus locks on face and is tracking correctly.
Check Arms & Head Framing Ensure arms and top of head are visible and within frame boundaries.
Head & Arm Recheck Re-verify framing after any position adjustment.
Framing Check in Footage Play back a short test clip to validate the final on-screen composition.
Chair & Actor Height Setup Adjust chair height and sitting position for consistent eye-line and posture.
Headspace Error — Investigate & Fix If top-of-head is clipped, adjust camera tilt or actor seating position.
Correct Framing Final Check Confirm final framing — check headspace, shoulders, and background.
Head & Hands Final Check Ensure hands and head are both clearly visible and not cropped.

🎙️ Audio Setup

Check XLR Mixer Levels Verify XLR input levels; aim for -12 dB to -6 dB on voice.
Recheck Replay Buffer Audio Ensure audio is included in the replay buffer output.
Check Replay Buffer Timeline with Claude Use Claude to review timeline integrity of replay buffer clips.
Retrieve Microphone from Boom Arm Position the boom arm mic at the correct distance and angle.
Audio Recording Spot-Check Record a test clip; play back to confirm clean, clear audio with no noise.

📝 Prompter & Script

Use Emojis & Emotions at Start in Prompter Add emotion cues and emojis to the opening lines for energy and engagement.
Prompter Shortcuts in Stream Deck Verify scroll speed, pause, and reset shortcuts are mapped on Stream Deck.
Duplicate Initial Lines for Easy Location Repeat Copy opening lines at the end so re-takes can start from a quick scroll.
Increase Prompter Font Size Set font to a comfortable reading size (typically 48-72px).
Font Size Verification Stand at recording position and confirm text is clearly readable at a glance.
Prompter Font Size Final Check Re-verify font size after any script edits or prompter layout changes.
Automate Prompter via Stream Deck Enable auto-scroll and test hands-free prompter operation through Stream Deck.
Updated Version — Add Prompter to Layout Confirm prompter window is integrated into the updated scene layout.
Add Direct Prompter Screen Button Add a direct one-tap Stream Deck button to jump to prompter fullscreen.
New Version — Prompter Ready Confirm the final prompter version is loaded and scroll-ready.

⚠️ Technical Verification (Framerate & Screens)

FPS Check — 30fps / Green Screen Error Confirm output is 30 fps; resolve green-screen artifact if visible.
Update Framerate in OBS Go to OBS settings → Video → set Common FPS to 30 (or project target).
Reset Framerate & Check for Dropped Frames Restart OBS after framerate change; monitor stats for dropped frames.
Record and Spot-Check Record 30 seconds and review immediately for framerate and quality issues.
FPS Multi-Record Check Record multiple short clips and verify FPS consistency across all of them.
All Screens Check Verify every monitor is showing the correct content — camera, prompter, OBS.

🤝 Script Review & Collaboration

Claude Script Generation & Review Use Claude to generate or refine script content for this session.
Script in Action — Dry Run Run through the full script once without recording to check flow and timing.
Share Script with Stakeholders Send the final script link to stakeholders for pre-production sign-off.
Update Font / Script & Verify with Parties Apply any last-minute font or script changes; confirm with all parties.
Dual Verification Two-person review of script and setup before hitting record.

✅ Post-Production & Wrap

Upload to Google Drive & Share Upload the final recording to Google Drive and share the link with the team.
Upload to YouTube (Unlisted) Upload the processed video to YouTube as unlisted for stakeholder review.
Green Screen Tidy-Up Roll up and store the green screen after filming is complete.
Greenscreen Lighting Tidy-Up Turn off and pack away all green screen lights safely.
Main Light & Boom Arm Tidy-Up Switch off key light and store boom arm microphone properly.
